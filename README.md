# Web-Based Portfolio Management System

Description
This project is a dynamic portfolio website built using PHP and MySQL.  
It allows an administrator to manage portfolio projects through a dashboard.

Features
- Admin Dashboard
- Add Projects
- Edit Projects
- Delete Projects
- View Portfolio Projects
- Public Portfolio Page

Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP

Installation
1. Download the project
2. Place it in `htdocs` folder in XAMPP
3. Import `portfolio_db.sql` into phpMyAdmin
4. Run the project:
http://localhost/portfolio-system/admin/admindashboard.php
